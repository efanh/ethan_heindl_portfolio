My history with Computer Science

Hi, I'm Ethan and I'm a senior Computer Science student at Longwood University. Ever since I can remember, I've always had a passion for anything related to computers. I was always fascinated by how all the different parts work and interact together, how the computer seemed to magically just run any software applications I downloaded, and how the computer was able to use all of that to connect to the internet, allowing for what seemed like endless opportunities.

That is why I decided to study computer science. I wanted to learn exactly what made all the different aspects work and how they worked. While taking all the different programming courses, I became even more interested in computers than I once already was. Learning how much truly went into creating the different computer programs I loved really put it into perspective that no matter how complex a program was, there was always going to be consistent innovation that could make that same program better and better as time went on. I began to realize the endless opportunities that computers have.

Of all the courses I've taken, the one that stuck out to me most was the course I took on web development. Web development allowed me continue coding, but also gave me the opportunity to throw in my own creative ideas, which I loved. I was able to create a webpage, and then create my own designs to really personalize exactly how I wanted it to look. I began using my own time outside of schoolwork to learn more about Javascript, CSS, and HTML, as well as learn about different frameworks such as Bootstrap to better help my designs.
